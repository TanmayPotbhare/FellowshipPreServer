Basic Information
  1. After you press submit on Section 5 you will not be able to edit anything.
  2. On Each section of the form you will see three buttons - Previous | Save & Next | Next | फॉर्म 5 विभागांमध्ये विभागलेला आहे
  3. Please select Save & Next to save and go to the next section.
  4. Please select Previous to go to the previous page section without saving the details.
  5. Please select Next to go to the next page section without saving the details.
  6. Each field has Instructions this symbol and text which will tell you how to fill that particular field. Also some fields are dependent on other fields please look at instructions before filling
  7. If you fail to submit the form correctly you can request to reset the form on the given mail ID below.
Information about Application Form Sections
  1. The form is divided into 5 sections | फॉर्म 5 विभागांमध्ये विभागलेला आहे
  2. Section 1 contains Personal Details | विभाग १ मध्ये वैयक्तिक तपशील आहेत
  3. Section 2 contains Qualification Details | विभाग २ मध्ये पात्रता तपशील आहेत
  4. Section 3 contains Domicile, Caste, Validity Details | विभाग ३ मध्ये अधिवास, जात, वैधता तपशील आहेत
  5. Section 4 contains Parents & Bank Details | विभाग 4 मध्ये पालक आणि बँक तपशील आहेत
  6. Section 5 contains Documents which you need to uplaod | विभाग 5 मध्ये तुम्हाला अपलोड करणे आवश्यक असलेले दस्तऐवज आहेत
